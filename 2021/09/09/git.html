Git安装

安装git

git config –global user.name “your name”

git config –global user.email “your email”

Git Bash生成SSH公钥

ssh -keygen -t rsa -C “您的邮箱地址”

cat ~/.ssh/id_rsa.pub 将文件打印至终端

上传

创建一个本地的版本库（其实也就是一个文件夹）

打开文件夹

git bash here

输入命令 git init

生成 .git文件夹 它是Git用来跟踪和管理版本库的 默认隐藏

把项目粘贴到这个本地Git仓库里面

git status 查看你当前的状态

git add . 该目录下的所有文件添加到仓库

git commit -m “shuoming”
