安装git

git config –global user.name “your name”

git config –global user.email “your email”

Git Bash生成SSH公钥

ssh -keygen -t rsa -C “您的邮箱地址”

cat ~/.ssh/id_rsa.pub 将文件打印至终端

上传

创建一个本地的版本库（其实也就是一个文件夹）

打开文件夹

输入命令 git init

把项目粘贴到这个本地Git仓库里面
